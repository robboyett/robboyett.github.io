ai-share-button

A platform-agnostic web component that extracts page content as Markdown and sends it to AI tools. Drop one <script> tag, place <ai-share-button> anywhere.

Installation

<!-- Turndown for DOM → Markdown conversion -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/turndown/7.1.3/turndown.min.js"></script>

<!-- The component -->
<script src="ai-share-button.js" type="module"></script>

<!-- Place it anywhere -->
<ai-share-button selector="article" label="Share to AI"></ai-share-button>

Attributes

selector "main" CSS selector for the content to capture
label "Share to AI" Text shown on the trigger button
options "copy,claude,chatgpt" Comma-separated list of enabled actions
char-limit 2000 Max characters passed via URL (Claude/ChatGPT)

CSS custom properties

All visual properties are exposed as CSS custom properties on :host, so you can theme it to match any site without touching Shadow DOM internals. 

ai-share-button {
  --ai-share-bg:      #0f0f0f;
  --ai-share-fg:      #e2e2e2;
  --ai-share-accent:  #7fff7f;
  --ai-share-border:  #2a2a2a;
  --ai-share-font:    "JetBrains Mono", monospace;
  --ai-share-radius:  6px;
}

How it works

When you trigger Copy or open an AI tool, the component finds the element matching your selector, converts its inner HTML to clean Markdown using Turndown, strips noise (script, style, nav, footer), then either copies to clipboard or opens the AI tool's URL with the content pre-loaded as a prompt.

Turndown is loaded lazily if not already present on the page — no build step required. The component uses Shadow DOM so your page styles never bleed in or out.
